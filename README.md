# Java Programs ☕

OOPs concepts implemented in Java from my BCA course.

## Programs

### 1. Student.java — Classes & Objects
- Creates Student objects with USN, name, branch, phone and percentage
- Demonstrates class creation and object arrays

### 2. BankAccount.java — Encapsulation
- Private variables with getters and setters
- Deposit and withdrawal with validation
- Demonstrates data hiding
